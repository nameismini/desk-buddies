# DeskBuddies — 바탕화면 캐릭터 런처

윈도우 바탕화면에 두 캐릭터가 떠 있고, 클릭하면 각자의 기능이 열립니다.

- 🧑 **트레이더 (남자)** — 따로 만들고 있는 **파이썬 자동매매 프로젝트를 실행**하는 버튼
- 👩 **애널리스트 (여자)** — **실시간 시세 + 경제뉴스 + 챗** 대시보드

---

## 1. 처음 한 번만: Node.js 설치

이 앱은 Electron으로 만들어졌어요. Node.js가 필요합니다.

1. https://nodejs.org 에서 **LTS 버전** 설치
2. 설치 후 PowerShell(또는 명령 프롬프트)을 새로 엽니다.

## 2. 실행

이 폴더(`런처`)에서 PowerShell을 열고:

```powershell
npm install      # 처음 한 번만 (electron 내려받기, 몇 분 걸림)
npm start        # 앱 실행
```

실행하면 화면 오른쪽 아래에 두 캐릭터가 떠요.

- 캐릭터에 마우스를 올리면 말풍선이 뜨고, **클릭하면 패널이 열립니다.**
- 캐릭터 위쪽 빈 공간을 **드래그하면 위치 이동**.
- 오른쪽 위 ⚙ 설정 / ✕ 종료.

## 3. 애널리스트(시황 챗) 설정

여자 캐릭터를 클릭 → 우측 상단 ⚙ →

- **LLM 제공자 / 모델 / API 키** 입력 후 저장하면 챗이 동작합니다.
  - Anthropic 예: 모델 `claude-haiku-4-5-20251001`
  - OpenAI 예: 모델 `gpt-4o-mini`
- **시세 마켓**: 업비트 코드(`KRW-BTC,KRW-ETH,KRW-SOL` …)
- 시세는 업비트 공개 API라 키 없이 바로 나오고, 뉴스도 자동으로 불러옵니다.
- 챗에 물으면 현재 시세 + 뉴스 헤드라인을 참고해서 답합니다.

> API 키는 외부로 새지 않고 이 PC의 설정 파일(`config.json`)에만 저장돼요.

## 4. 트레이더(자동매매) 연동

남자 캐릭터를 클릭 →

- **파이썬 실행 파일**: 보통 `python` 또는 `py`
- **자동매매 진입 스크립트 경로**: 예) `C:\projects\autotrade\main.py`
- **실행 인자**(선택): 예) `--mode live`
- 저장 후 **▶ 자동매매 실행**을 누르면 그 파이썬 프로젝트를 새 프로세스로 띄웁니다.

런처는 **실행만** 담당해요. 매매 로직·거래소 API 키·리스크 관리는 파이썬 프로젝트 안에서 처리하세요.

⚠️ **실거래 연결 전 주의**: 실제 돈이 오가는 자동매매는 버그 한 줄로 손실이 날 수 있어요. 모의매매 → 소액 실거래 순으로 충분히 검증한 뒤 연결하시길 권합니다.

## 5. (선택) exe로 만들기

설치 없이 더블클릭으로 쓰고 싶으면:

```powershell
npm run dist
```

`dist/` 폴더에 포터블 exe가 생성됩니다.

## 파일 구조

| 파일 | 역할 |
|---|---|
| `main.js` | Electron 메인 — 창 생성, 파이썬 실행, 뉴스 수집, LLM 호출 |
| `preload.js` | 렌더러 ↔ 메인 안전한 통신 브릿지 |
| `overlay.html` | 바탕화면에 떠 있는 캐릭터 |
| `dashboard.html` | 애널리스트(시황·뉴스·챗) 패널 |
| `trader.html` | 트레이더(자동매매 실행) 패널 |
| `assets/` | 캐릭터 이미지 |

캐릭터 이미지를 바꾸고 싶으면 `assets/male_2.png`, `assets/female_2.png`를 교체하세요.

> 캐릭터는 가만히 있을 때 살짝 숨쉬듯 움직이고(idle), 말풍선이 뜨면 까딱까딱 말하는 애니메이션이 재생됩니다.
> 또 설정한 마켓의 시세가 **15초 사이 일정 폭(기본 0.8%) 이상 급변**하면 애널리스트가 말풍선 + OS 알림으로 **먼저 알려줍니다.** 민감도는 대시보드 ⚙ 설정에서 조절하세요.
